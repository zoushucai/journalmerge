AbbrevIso API

This simple API provides abbreviations of journal titles attempting to implement the ISO-4 standard. It executes the AbbrevIso library (GitHub). See the browser UI for details and caveats.

  • GET /a/Journal%20of%20Foo
    → J. Foo
  • GET /a/Zeitschrift%20f%C3%BCr%20deutsches%20Altertum%20und%20deutsche%20Literatur
    → Z. dtsch. Altert. dtsch. Lit.

Every query must provide a non-empty User-Agent HTTP header.
